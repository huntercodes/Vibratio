# Vibratio

Custom music player for iOS!

User can login to their Spotify account, listen to their own playlists, create/edit their own playlists, search Spotify's entire catalog, and more!


### Technologies Used

- Swift
- UIKit
- Spotify APIs
- SDWebImage
- Firebase
- Appirater
